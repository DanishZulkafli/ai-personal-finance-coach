# Smart Study Planner

Smart Study Planner is a simple interactive web application that helps users organize study tasks based on priority and study duration.

This project is built using HTML, CSS, and JavaScript. It is designed as a beginner-friendly productivity tool for students, developers, and lifelong learners.

## Features

- Add a study task
- Select task priority
- Select study duration
- Generate a suggested study plan
- Display focus mode, break time, and strategy
- Clean and responsive user interface

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

The user enters a study task, selects the priority level, and chooses a study duration. The application then generates a recommended study plan based on the selected inputs.
